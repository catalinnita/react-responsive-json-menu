# react-responsive-json-menu
A simple responsive menu created from a json file

-------------

Install:

- clone the repo
- npm install [ in cloned directory ]
- npm start [ in cloned directory ]
- go to http://localhost:8080/

-------------

	- latest firefox + mobile emulation
	- latest edge
